# **NihonGo**

#### 제공기능 명세
1. 내가 지정한 환율 도달 시 알림카톡
2. 소비세 8%를 고려한 실제지불금액 계산
3. 동전을 최소화하는 지불방법 추천

------

#### 사용 기술스택
Ruby on Rails Framework, HTML/HTML5, CSS/CSS3, Javascript, AWS Cloud9, CSS Framework(Bootstrap, Materialize)
